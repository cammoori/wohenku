# 乡镇工作通讯录应用

这是一个专属于乡镇工作的通讯录应用，支持网页版、桌面版和Android版，可在不同设备上使用。

## 功能特点

- **单位分类管理**：科室、村庄等分类管理
- **人员信息管理**：添加、编辑、删除人员信息
- **管理关系图**：部门层级关系和管理关系可视化
- **搜索和筛选**：快速查找人员信息
- **导出功能**：导出通讯录为CSV格式
- **数据持久化**：本地存储，数据不丢失
- **语音转文字**：语音输入转文字功能
- **图片转文字识别**：OCR图片识别功能
- **八卦板块**：乡镇新闻和动态，支持点赞、评论、表情包和外号功能
- **亲属板块**：人员亲属关系管理
- **用户系统**：注册登录，管理员和普通用户权限管理
- **应用下载**：自动检测设备并提供对应版本下载
- **响应式设计**：适配电脑和移动设备

## 安装和运行

### 网页版
1. **直接访问**：在浏览器中打开 `index.html` 文件
2. **下载应用**：点击顶部的"下载应用"按钮，自动检测设备并提供对应版本下载

### 桌面应用（Windows、macOS、Linux）

#### 方法一：压缩包安装（推荐）
1. **下载压缩包**：从网页版或构建产物中获取对应平台的压缩包
2. **解压压缩包**：将压缩包解压到任意目录
3. **运行应用**：双击解压目录中的可执行文件即可运行

#### 方法二：从源码构建
1. **安装依赖**
   ```bash
   npm install
   ```

2. **运行开发版本**
   ```bash
   npm start
   ```

3. **构建压缩包**
   ```bash
   npm run build-zip
   ```
   构建完成后，压缩包会生成在 `dist` 目录中。

### Android应用（红米K90等设备）

#### 方法一：直接安装APK（推荐）
1. **下载APK文件**：从网页版或构建产物中获取APK文件
2. **安装到设备**：将APK文件传输到红米K90设备并安装

#### 方法二：从源码构建
1. **安装Capacitor CLI**
   ```bash
   npm install -g @capacitor/cli
   ```

2. **添加Android平台**
   ```bash
   npx cap add android
   ```

3. **同步到Android平台**
   ```bash
   npx cap sync
   ```

4. **打开Android Studio**
   ```bash
   npx cap open android
   ```

5. **构建APK**
   - 在Android Studio中，点击"Build" > "Build Bundle(s) / APK(s)" > "Build APK(s)"
   - 构建完成后，APK文件会生成在 `android/app/build/outputs/apk/debug/` 目录中

6. **安装到设备**
   - 连接红米K90设备
   - 将APK文件传输到设备并安装

## 使用指南

### 基本操作
- **添加人员**：点击顶部的"添加"按钮，填写人员信息
- **编辑人员**：点击人员列表项，在详情页点击"编辑"按钮
- **删除人员**：点击人员列表项，在详情页点击"删除"按钮
- **搜索人员**：在搜索框中输入关键词
- **筛选分类**：点击分类标签筛选人员
- **导出通讯录**：点击顶部的"导出"按钮

### 用户系统
- **注册**：点击顶部的"登录"按钮，切换到注册标签
- **登录**：点击顶部的"登录"按钮，输入用户名和密码
- **管理员账号**：默认管理员账号为 123 / 233
- **权限管理**：管理员可以修改所有内容，普通用户只能修改自己的内容
- **注册限制**：管理员账号无法通过注册获取，只能使用默认账号

### 语音转文字
- 点击顶部的"语音"按钮
- 点击开始录音按钮
- 等待识别完成后，可复制识别结果

### 图片转文字识别
- 点击顶部的"OCR"按钮
- 选择要识别的图片
- 点击"识别"按钮
- 查看识别结果

### 八卦板块
- 点击底部导航栏的"八卦"图标
- 浏览乡镇动态
- **点赞**：点击心形图标为动态点赞
- **评论**：点击评论图标查看和发布评论
- **添加八卦**：点击"添加八卦"按钮发布新动态
- **表情包**：添加动态时可以插入表情包
- **外号**：添加动态时可以设置自己的外号
- **关联人员**：可关联相关人员

### 亲属板块
- 点击底部导航栏的"亲属"图标
- 查看亲属关系图
- 浏览亲属列表
- 亲属信息与人员信息关联

### 应用下载
- 点击顶部的"下载应用"按钮
- 选择"自动下载"或手动选择设备类型
- 下载对应版本的应用安装包

## 技术栈

- **前端**：HTML、CSS、JavaScript、Tailwind CSS、Font Awesome、ECharts
- **桌面应用**：Electron
- **移动应用**：Capacitor
- **数据存储**：localStorage

## 数据同步

- **本地存储**：数据存储在设备本地，确保隐私安全
- **跨设备同步**：可通过导出/导入CSV文件实现数据同步
- **备份恢复**：定期导出CSV文件作为备份

## 注意事项

- 桌面应用支持Windows、macOS、Linux平台
- Android应用支持Android 7.0及以上版本
- 网页版支持所有现代浏览器
- 首次运行时会使用默认数据，后续修改会自动保存
- 语音和OCR功能需要设备支持相应硬件
- 管理员账号具有最高权限，请妥善保管
